# atelier-samui

Simple static website for atelier-samui.com with multilingual support.

## Supported languages

The site automatically chooses a language using this order:
1. Saved preference in localStorage.
2. Browser language (`navigator.languages` / `navigator.language`).
3. Time-zone based location fallback.
4. English default.

Available languages:
- English
- French
- German
- Russian
- Thai
- Arabic

## Run locally

Open `index.html` in a browser.
